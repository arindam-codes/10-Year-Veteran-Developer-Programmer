# Roblox Builder Asset Library

Reusable modular assets created for fast Roblox game development.

Goal: Build a library of reusable building kits that can be used across multiple games.

---

# 1. Lobby Kit

Reusable spawn and lobby environment.

Components:
- Spawn platform
- Leaderboard wall
- Teleport pads
- Shop area
- Display boards
- Social hangout area
- AFK zone

Purpose:
Used in almost every Roblox game as the main entry area.

---

# 2. Obstacle Kit (Obby System)

Reusable obstacle pieces.

Components:
- Moving platforms
- Rotating bars
- Jump pads
- Disappearing platforms
- Lava blocks
- Speed pads
- Checkpoint models

Purpose:
Used in obby games, minigames, and challenge maps.

---

# 3. Arena Map Pack

Reusable combat arenas.

Maps:
- Small PvP arena
- Battleground arena
- Tournament stage
- Rooftop arena
- Training arena

Purpose:
Used for combat games and fighting systems.

---

# 4. Simulator Map Template

Base template for simulator-style games.

Components:
- Coin farming areas
- Egg shop area
- Upgrade shop area
- Rebirth portal
- World teleport portals
- Leaderboard area

Purpose:
Used for clicker, pet, grinding, and simulator games.

---

# 5. Tycoon Asset Pack

Reusable tycoon building pieces.

Components:
- Conveyor belts
- Droppers
- Upgrade buttons
- Factory walls
- Collect buttons
- Cash collectors

Purpose:
Used in tycoon and factory-style games.

---

# 6. NPC Character Pack

Reusable NPC characters.

NPC Types:

- Enemy NPC

- Guide NPC

Purpose:
Used in RPG, adventure, and simulator games.

Weapons:

- Magic staff

Purpose:

# 8. Environment Asset Pack

Assets:

Used for indoor environments.

Reusable visual effects.

Effects:

# 11. UI Layout Kit

Layouts:

Animations:

Purpose:

---

# 13. Sound Pack

Reusable sound effects.

Sounds:
- Hit sound
- Coin pickup
- Level up
- UI click
- Explosion

Purpose:
Used across all games.

---

# 14. Lighting Presets

Reusable lighting environments.

Presets:
- Bright simulator lighting
- Dark horror lighting
- Sunset lighting
- Neon lighting

Purpose:
Used to quickly style different game types.

---

# Goal

Build a reusable asset library so that new Roblox games can be created faster by combining pre-built systems and assets.
