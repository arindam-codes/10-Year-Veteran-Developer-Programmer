# Roblox Animator Skill Tree (Market Analysis)

## 1. Core Animation Fundamentals (Foundation)

Every job requires these.

### Animation Principles

These come from classic animation theory:

* Timing
* Spacing
* Weight
* Anticipation
* Follow-through
* Overlap
* Ease in / ease out
* Exaggeration
* Pose readability
* Motion arcs

These are **what separates amateur animation from professional animation**.

Example requirement from posts:

> "proper easing, smooth motion, believable weight"

---

# 2. Roblox Rig Systems

Animators must understand different rigs.

### R6

Classic Roblox rig.

Used in:

* older games
* combat games
* battleground games

Skills required:

* limited joints
* exaggerated movement

---

### R15

Modern Roblox rig.

Used in:

* most new games
* UGC emotes
* RPG games

Skills required:

* smoother movement
* full body animation
* better pose control

---

### Custom Rigs

Very common in high-budget games.

Examples:

* creatures
* monsters
* animals
* anime characters

Skills required:

* custom bone structures
* unique joint limits
* mesh deformation
